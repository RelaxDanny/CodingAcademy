 1. 자동차 하이패스 
 2. 옷가게 문앞에서 (도둑질 햇을때 삐삐소리)
 3. 공장안에 각 식품별 태그 인식


RFID의 사용법: feat. arudino
1. RFID 리더기(장바구니) 음식과 식품들을 인식한다.
2. 인식한 식품들의 고유넘버(값)을 받고 예시: XXFF01001010
3. 아두이노를 통해서 받은 데이터를 python으로 보내. 
4. python 받은 XXFF01001010을 서버에 보내
5. 서버에서 받자마자 어떤 제품인지를 Database에서 골라온다. 
6. 서버에서 클라이언트(고객의 휴대폰 바로마켓앱)에 Database 가져온 정보를 쏜다.

---Database---

[XXFF01001010] => [오렌지] : [유통기한: 2020년20월20일] : [개수: 30개] : [가격: 30,000원]

7. 바로마켓 앱에 리스트 형식으로 작성한다.
8. 고객이 결제를 완료하면 RFID의 센서에 그 장바구니가 인식이 안되도록 해야함.
9. 옷가게에서 결제안하고 나가면 삐소리 나는 원리 공부해보기.
10. 고객 결제.
11. 고객의 결제가 되면 서버에 결제라는 boolean(기존 FALSE)칸에다 True값을 전송하고 True값을 받은 데이터베이스는
서버로 부터 다시 RFID 리더기에 전송된다 (리더기의 한계점 찾아보기). 리
더기는  True값을보면 삐소리 안내기

**database분류 장바구니의 RFID 고유넘버대로 database 분류. -> 모든 제품이 TRUE값일때만 결제

EXCEPTION: RFID 태깅이 안된 제품들 또는 낱개의 과일들. 
Solution : 상인이 직접 바로마켓 상인전용 앱을 통해 판매 할 과일의 가격과, 개수, 이름을 넣고 서버에 전송.
서버에서는 고객 바로마켓 앱에 전송.
